# iced_helper

A small set of components designed to simplify working with Iced. These are primarily UI elements and components that make it easier to work with subscriptions for creating processes that illustrate the progress of their work.

# Use

To connect, use: 
```toml
iced_helper = {git = "https://github.com/Andrewkoro105/iced_helper.git"}
```

To view the documentation conveniently, use: 
```sh
cargo doc --open
```
